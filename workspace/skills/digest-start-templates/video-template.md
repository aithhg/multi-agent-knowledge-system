## 消化切入问题

### 内容定性

- [ ] **这是什么领域的？**
  - [ ] AI/机器学习
  - [ ] 商业/创业
  - [ ] 哲学/思想
  - [ ] 技能/工具
  - [ ] 心理学/认知
  - [ ] 其他：____

- [ ] **作者/演讲者的核心观点是什么？**（1-2句话）

- [ ] **这内容的目标受众是谁？**
  - [ ] 入门新手
  - [ ] 从业者
  - [ ] 资深专家
  - [ ] 泛大众

### 关键提取

- [ ] **最颠覆认知的一个点**是什么？

- [ ] **最实用的一个操作点**是什么？

- [ ] **与已有关联的知识有哪些？**
  - [[已有概念卡 1]]
  - [[已有概念卡 2]]

### 消化方向候选

根据内容选择最合适的 1-2 个消化方向：

| 方向 | 适用场景 |
|------|---------|
| **方向A：方法论框架** | 内容有系统性方法/框架/流程 |
| **方向B：案例/故事** | 内容以真实案例为主，有启发性故事 |
| **方向C：反面教材** | 内容揭示常见误区/错误做法 |
| **方向D：工具/资源** | 内容推荐了实用工具/资源/链接 |
| **方向E：概念澄清** | 内容澄清了一个常见误解或概念 |

## 消化执行检查

完成消化后检查：

- [ ] 是否生成了结构化摘要？
- [ ] 是否提取了概念卡？（如有必要）
- [ ] 是否更新了主题页/索引？
- [ ] 是否触发了同步到 D&H&R/？

---

## 模板使用说明

1. 复制本模板到消化任务中
2. 先回答「内容定性」问题，建立基本理解
3. 再回答「关键提取」问题，找到最有价值的点
4. 选择一个「消化方向」，聚焦消化
5. 消化完成后执行检查清单
