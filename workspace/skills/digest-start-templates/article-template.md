# 文章类内容消化切入问题清单

> 适用：公众号文章/博客/论文/新闻/长帖等文本内容

---

## 消化切入问题

### 来源判断

- [ ] **这是一手原创还是二手解读？**
  - [ ] 一手原创（作者本人的研究和观点）
  - [ ] 二手解读（对已有内容的整理和转述）
  - [ ] 混合型

- [ ] **作者背景是什么？**（影响可信度）
  - 职业/身份：____
  - 领域积累：____
  - 可能的立场/偏见：____

### 核心提炼

- [ ] **标题承诺了什么？**（标题的字面意思）

- [ ] **文章实际证明了什么？**（与标题的匹配度）
  - [ ] 完全匹配
  - [ ] 部分匹配
  - [ ] 偏离较大

- [ ] **最有说服力的证据/数据是什么？**

- [ ] **最核心的一个观点是什么？**（能否用一句话概括）

### 批判性思考

- [ ] **作者的假设是什么？**（未被证明但被视为理所当然的前提）

- [ ] **有什么被忽略的反面证据？**（与作者观点相悖但未被提及）

- [ ] **这个结论的适用边界在哪里？**（什么时候成立，什么时候不成立）

### 关联整合

- [ ] **与已有关联的知识有哪些？**
  - [[已有概念卡 1]]
  - [[已有概念卡 2]]

- [ ] **这篇文章能补充/修正/挑战已有的哪些观点？**

### 消化方向候选

根据内容选择最合适的 1-2 个消化方向：

| 方向 | 适用场景 |
|------|---------|
| **方向A：观点萃取** | 文章核心是作者的一个/多个观点 |
| **方向B：事实记录** | 文章核心是事件/数据的记录 |
| **方向C：方法分享** | 文章是一篇教程/指南/方法论 |
| **方向D：产品分析** | 文章是对某个产品/服务的分析 |
| **方向E：现象解读** | 文章是对某个社会/行业现象的解读 |

## 消化执行检查

完成消化后检查：

- [ ] 是否生成了结构化摘要？
- [ ] 是否提取了概念卡？（如有必要）
- [ ] 是否更新了主题页/索引？
- [ ] 是否触发了同步到 D&H&R/？

---

## 模板使用说明

1. 复制本模板到消化任务中
2. 先做「来源判断」，建立对可信度的基本认知
3. 再做「核心提炼」，抓住文章最核心的内容
4. 带着「批判性思考」问题读第二遍
5. 选择一个「消化方向」，聚焦消化
6. 消化完成后执行检查清单
