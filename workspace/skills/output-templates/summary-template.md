# 摘要模板

> 适用：结构化摘要，用于快速了解原内容要点

---

## 模板

```yaml
---
id: summary-YYYYMMDD-slug
title: 摘要标题
source_id: raw文件id
created_at: YYYY-MM-DD
type: summary
topics: []
concepts: []
---

## Core Idea

<!-- 一句话概括最重要的观点，不超过30字 -->

## Key Points

1. <!-- 第一个要点，1-2句话 -->

2. <!-- 第二个要点，1-2句话 -->

3. <!-- 第三个要点，1-2句话 -->

## Evidence

<!-- 支撑观点的关键事实/数据/引用，使用 - 列表 -->

## Open Questions

<!-- 读完后仍然存在的疑问，1-3个 -->

## Related

<!-- 与已有关知识库的关联 -->
- [[相关概念卡1]]
- [[相关概念卡2]]

## Source

<!-- 原始材料链接 -->
- [[raw文件id]]
```

---

## 长度规范

| 部分 | 长度 | 说明 |
|------|------|------|
| Core Idea | 20-30 字 | 一句话 |
| Key Points | 50-150 字/条 | 1-3条，每条1-2句话 |
| Evidence | 50-200 字 | 核心证据 |
| Open Questions | 30-100 字 | 1-3个问题 |
| **总计** | 200-500 字 | |

---

## 必须包含

- [ ] Core Idea 用一句话概括
- [ ] Key Points 列出 2-5 个要点
- [ ] 每个要点有具体内容，不是泛泛而谈
- [ ] Related 链接到已有关联知识
- [ ] Source 指向原始 raw 文件

## 禁止出现

- ❌ 标题党的夸大表述
- ❌ 没有证据支撑的观点
- ❌ 与原内容重复率超过 50% 的「摘要」
